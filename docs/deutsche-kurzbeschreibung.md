# Deutsche Kurzbeschreibung

Der **AI-based Offergenerator for Industrial Projects** automatisiert die Erstellung professioneller PowerPoint-Angebote. Nutzende geben Projektstammdaten, Ausgangssituation, Projektziele und Arbeitspakete in eine Browseroberfläche ein. Das System erweitert die Inhalte KI-gestützt, wählt die passende Vorlage und erzeugt automatisch eine gebrandete PPTX-Datei.

Dieses öffentliche Showcase-Repository zeigt Architektur, Workflows und Diagramme, ohne den proprietären Quellcode offenzulegen.

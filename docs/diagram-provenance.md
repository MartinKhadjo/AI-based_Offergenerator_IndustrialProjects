# Diagram Provenance

The diagrams in `docs/assets/` were derived from the provided software manuals and were included as clean, readable excerpts for documentation purposes.

## Included assets
- `processing-flow.png` – extracted from the English offer-generator manual, section **4.1 Processing Flow**, Figure 6.
- `form-submission-sequence.png` – extracted from the English offer-generator manual, section **4.2 Sequence Diagram (Form Submission)**, Figure 7.
- `component-hierarchy.png` – extracted from the English offer-generator manual, section **4.3 Component Hierarchy**, Figure 8.
- `browser-ui.png` – cropped from the user-guide section as a supporting UI illustration.

## Source manuals used
- `PEM_Offer_Generator_Manual_and_Documentation_EN.pdf`
- `PEM_Offer_Generator_Handbuch_und_Dokumentation_DE.pdf` (cross-reference only)

The public showcase keeps the diagrams visually faithful to the manuals rather than redrawing them from scratch.
